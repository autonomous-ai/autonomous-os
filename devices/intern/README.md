# Autonomous Intern

**Text it. It's handled.** A plug-and-play AI device that automates your busywork while
your laptop sleeps — the second reference device for [Autonomous](../../README.md).

🔗 **Product page:** https://www.autonomous.ai/ai-gadgets/intern

![Autonomous Intern](images/intern.webp)

## What it is

A compact computer that runs AI models locally and works for you over the chat apps you
already use — Telegram, Slack, Discord. Zero setup; it lives on your network, not your
laptop. Email triage, GitHub monitoring, scheduling, web scraping, booking, billing.

## Capabilities

**Lamp minus the body** — audio, vision, sensing. No motion, no display. The same OS image
as Lamp, declaring fewer capabilities — not a fork. Declared in [`DEVICE.md`](DEVICE.md).

## Memory & privacy

Remembers API keys, project context, your writing style, and team vocabulary — stored
locally as files. No cloud dependency; everything stays on the device.
